# 09 - Function Challenges

This challenge lesson practices functions, arrow functions, return values, object returns, spread syntax, and IIFEs.

## Files

- `index.html` loads the challenge page and includes `script.js`.
- `script.js` contains implemented solutions for the challenges.
- `08-function-challenges.md` contains the original challenge instructions, hints, and solution notes.

## What This Covers

- Converting Fahrenheit to Celsius with a function.
- Writing a one-line arrow function.
- Formatting a result with a template literal.
- Finding the minimum and maximum values in an array.
- Using `Math.min()`, `Math.max()`, and spread syntax.
- Returning an object from a function.
- Creating an IIFE that calculates and logs rectangle area immediately.

## Things To Notice

The final challenge uses an IIFE, so the rectangle area message logs as soon as the script loads. No separate function call is needed after the IIFE.
