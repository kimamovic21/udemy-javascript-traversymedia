# 01 - Function Basics

This lesson introduces how to define, call, and return values from JavaScript functions.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the function basics examples.

## What This Covers

- Defining a function with the `function` keyword.
- Calling, invoking, or executing a function.
- Passing values into a function.
- The difference between parameters and arguments.
- Returning values with `return`.
- Assigning a returned value to a variable.
- Understanding that code after `return` does not run.

## Things To Notice

`add()` logs its result directly, while `subtract()` returns a value. Returning is more flexible because the caller can store, log, or reuse the result.
