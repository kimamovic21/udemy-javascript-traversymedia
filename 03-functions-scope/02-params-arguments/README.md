# 02 - Params and Arguments

This lesson goes deeper into function inputs by using default parameters, rest parameters, objects, and arrays.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains examples of different ways to pass data into functions.

## What This Covers

- Setting default parameter values.
- Returning strings from functions.
- Collecting many arguments with rest parameters, `...numbers`.
- Looping through rest parameter values.
- Passing an object into a function.
- Passing an object literal directly into a function call.
- Passing an array into a function.
- Selecting a random item from an array.

## Things To Notice

Rest parameters turn separate arguments into an array. That is why `sum(1, 2, 3)` can loop over `numbers` like any other array.
