# 10 - Execution Context In Action

This lesson is designed for stepping through JavaScript execution in browser DevTools.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains a small example for observing how variables and function calls are executed.

## What This Covers

- Creating global variables.
- Defining a reusable function.
- Creating local variables inside a function.
- Returning a function result.
- Calling the same function with different arguments.
- Watching code run step by step with breakpoints.

## Things To Notice

Open the Sources tab in DevTools, add a breakpoint near the top of `script.js`, and step through the file. The goal is to see execution context behavior, not to build a visible page feature.
