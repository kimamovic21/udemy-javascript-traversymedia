# 11 - Call Stack

This lesson demonstrates the call stack by comparing separate function calls with nested function calls.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains call stack examples.

## What This Covers

- Defining multiple functions.
- Calling functions one after another.
- Using DevTools breakpoints to watch execution order.
- Understanding how nested function calls build up on the call stack.
- Comparing sequential calls with a function that calls another function.

## Things To Notice

The second example is commented out. Uncomment it when you want to watch `first()` call `second()`, then `second()` call `third()` in the DevTools call stack.
