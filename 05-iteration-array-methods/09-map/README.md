# 09 - Map

This lesson introduces the `map()` array method for creating a new array by transforming each item.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `map()` examples.

## What This Covers

- Doubling numbers with `map()`.
- Comparing `map()` with a manual `forEach()` approach.
- Creating an array of company names.
- Creating new object shapes from existing objects.
- Calculating company lengths from start and end years.
- Chaining multiple `map()` calls.
- Combining `filter()` and `map()` in one chain.

## Things To Notice

`map()` returns a new array with the same number of items as the original array. Use `filter()` first when you need to remove items before transforming them.
