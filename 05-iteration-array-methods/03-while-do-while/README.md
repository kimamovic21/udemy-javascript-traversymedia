# 03 - While Do While

This lesson introduces `while` and `do...while` loops as alternatives to `for` loops.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `while` and `do...while` examples.

## What This Covers

- Creating a `while` loop.
- Updating a counter manually inside the loop.
- Looping over an array with a `while` loop.
- Nesting `while` loops.
- Creating multiplication table style output with nested loops.
- Running a `do...while` loop at least once.

## Things To Notice

The same `i` counter is reused across examples, so its current value affects which loops run. Resetting `i` before an example is a good way to test each loop separately.
