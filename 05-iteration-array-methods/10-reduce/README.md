# 10 - Reduce

This lesson introduces the `reduce()` array method for combining many values into one final result.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `reduce()` examples.

## What This Covers

- Adding numbers together with `reduce()`.
- Using an accumulator and current value.
- Providing an initial accumulator value.
- Writing a shorter reducer with an arrow function.
- Comparing `reduce()` with a manual loop.
- Calculating a shopping cart total from object prices.

## Things To Notice

The second argument to `reduce()` sets the starting accumulator value. In these examples it starts at `0`, which makes sense for totals.
