# 08 - Filter

This lesson introduces the `filter()` array method for creating a new array with only the items that pass a test.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `filter()` examples.

## What This Covers

- Filtering numbers with a callback function.
- Writing shorter filter callbacks with arrow functions.
- Comparing `filter()` with a manual `forEach()` approach.
- Filtering objects by category.
- Filtering objects by start and end years.
- Finding companies that lasted a minimum number of years.

## Things To Notice

`filter()` keeps the original array unchanged and returns a new array. The callback should return `true` for items you want to keep.
