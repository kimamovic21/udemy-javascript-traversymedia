# 05 - For Of Loop

This lesson introduces the `for...of` loop for iterating directly over iterable values.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `for...of` examples.

## What This Covers

- Looping through arrays with `for...of`.
- Reading object values from an array of objects.
- Looping over each character in a string.
- Creating and filling a `Map`.
- Destructuring map entries into `key` and `value`.

## Things To Notice

`for...of` gives you the value from the iterable. In the map example, each value is a `[key, value]` pair, which is why array destructuring works nicely there.
