# 07 - forEach

This lesson introduces the `forEach()` array method for running a function once for every array item.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `forEach()` examples.

## What This Covers

- Calling `forEach()` on an array.
- Writing callback functions with regular function syntax.
- Writing callback functions with arrow function syntax.
- Accessing the item, index, and original array inside the callback.
- Passing a named function into `forEach()`.
- Looping over an array of objects.

## Things To Notice

`forEach()` is useful when you want to do something for each item, such as logging values. It does not create and return a new transformed array like `map()` does.
