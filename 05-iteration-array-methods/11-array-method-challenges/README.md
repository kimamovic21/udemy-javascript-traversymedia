# 11 - Array Method Challenges

This challenge set practices combining array methods to filter, transform, and reduce data.

## Files

- `11-array-method-challenges.md` contains the challenge instructions and expected results.
- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the challenge solutions.

## What This Covers

- Filtering people by age.
- Mapping filtered people into smaller objects.
- Combining first and last names into one `name` value.
- Filtering positive numbers.
- Reducing positive numbers into a total.
- Capitalizing words with `map()`, `toUpperCase()`, and `slice()`.

## Things To Notice

The first two challenges chain array methods, which keeps the data flow compact: filter the array down first, then map or reduce the remaining values into the final result.
