# 06 - For In Loop

This lesson introduces the `for...in` loop for iterating over object keys.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `for...in` examples.

## What This Covers

- Looping through object keys with `for...in`.
- Reading object values with bracket notation.
- Looping through array indexes with `for...in`.
- Using an index key to read array values.

## Things To Notice

`for...in` gives you keys or indexes, not the values directly. For arrays, `for...of` is usually cleaner when you only need the item values.
