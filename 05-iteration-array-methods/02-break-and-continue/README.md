# 02 - Break and Continue

This lesson demonstrates how to control loop execution with `break` and `continue`.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `break` and `continue` examples.

## What This Covers

- Stopping a loop early with `break`.
- Skipping the current loop iteration with `continue`.
- Checking loop values with an `if` statement.
- Comparing how loop output changes after `break` or `continue` runs.

## Things To Notice

`break` exits the loop completely when `i` reaches `15`, while `continue` only skips logging `13` and then lets the loop keep running.
