# 01 - For Loop

This lesson introduces the `for` loop and shows how to repeat code with a counter, condition, and increment step.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `for` loop examples.

## What This Covers

- Understanding the initial expression, condition expression, and increment expression.
- Running repeated statements inside a loop block.
- Using conditional logic inside a loop.
- Writing nested loops.
- Looping through an array by index.
- Reading array values with `names[i]`.

## Things To Notice

The active example loops through the `names` array and treats `John` differently from the other names. The earlier examples are commented out so you can uncomment them one at a time while practicing.
