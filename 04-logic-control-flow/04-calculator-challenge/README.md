# 04 - Calculator Challenge

This challenge practices control flow by building a small calculator function that chooses an operation based on an operator string.

## Files

- `04-calculator-challenge.md` contains the challenge instructions and example solution.
- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the calculator solution.

## What This Covers

- Defining a function with multiple parameters.
- Using a `switch` statement to choose between operations.
- Handling addition, subtraction, multiplication, and division.
- Returning a result from a function.
- Providing a fallback message for an invalid operator.
- Testing the function with different operator inputs.

## Things To Notice

The current function call passes `&`, so the `default` branch runs and returns the invalid operator message. Try replacing it with `+`, `-`, `*`, or `/` to test the other branches.
