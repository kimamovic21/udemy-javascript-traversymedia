# 07 - Logical Assignment

This lesson introduces logical assignment operators as shorter ways to conditionally update variables.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the logical assignment examples.

## What This Covers

- Assigning only when a value is falsy with `||=`.
- Assigning only when a value is truthy with `&&=`.
- Assigning only when a value is `null` or `undefined` with `??=`.
- Comparing logical assignment operators to longer `if` statements.
- Comparing logical assignment operators to regular logical expressions.

## Things To Notice

Logical assignment changes the variable in place. The examples show the longer form first so it is easier to see what each shorthand operator replaces.
