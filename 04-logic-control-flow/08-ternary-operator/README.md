# 08 - Ternary Operator

This lesson introduces the ternary operator as a compact way to write simple conditional expressions.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the ternary operator examples.

## What This Covers

- Rewriting a simple `if` and `else` statement as a ternary expression.
- Returning one of two values from a condition.
- Assigning conditional results to variables.
- Running multiple expressions with the comma operator.
- Using `&&` as a shorter pattern when no `else` action is needed.

## Things To Notice

Ternaries are useful for short decisions, but longer branches can become harder to read. The dashboard redirect example shows both the compact style and why readability still matters.
