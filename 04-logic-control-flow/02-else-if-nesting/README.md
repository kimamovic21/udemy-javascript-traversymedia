# 02 - Else If Nesting

This lesson expands conditional logic with `else if` branches, nested `if` statements, and combined conditions.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the `else if`, nested condition, and logical operator examples.

## What This Covers

- Checking multiple possible outcomes with `if`, `else if`, and `else`.
- Reading the hour from a `Date` object.
- Nesting one `if` statement inside another.
- Running more specific checks inside broader conditions.
- Combining comparisons with the logical AND operator `&&`.
- Allowing alternate matches with the logical OR operator `||`.

## Things To Notice

The date is created with a fixed hour, so changing the hour value is an easy way to test each branch and see different console messages.
