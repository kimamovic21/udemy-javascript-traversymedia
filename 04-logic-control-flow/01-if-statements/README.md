# 01 - If Statements

This lesson introduces `if` statements and shows how JavaScript runs different blocks of code based on boolean conditions.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the conditional statement examples.
- `comparison-operators.png` is a reference image for comparison operators.

## What This Covers

- Writing basic `if` statement syntax.
- Evaluating expressions with comparison operators.
- Using `else` when the original condition is false.
- Comparing values with strict equality and inequality.
- Seeing how block scope affects variables declared inside an `if` block.
- Writing shorthand `if` and `else` statements.

## Things To Notice

The `console.log(z)` line throws an error because `z` is declared inside the `if` block. That is intentional here: it demonstrates that `const` and `let` are block scoped.
