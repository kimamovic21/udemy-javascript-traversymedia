# 03 - Switches

This lesson introduces `switch` statements as another way to choose between multiple possible code paths.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the switch statement examples.

## What This Covers

- Creating a `switch` statement.
- Matching immediate values with `case`.
- Stopping execution in a case with `break`.
- Handling unmatched values with `default`.
- Using `switch (true)` to evaluate ranges.
- Comparing switch-based branching with `if` and `else if`.

## Things To Notice

JavaScript months are zero-based when using `Date`, so `new Date(2022, 1, 10)` represents February even though the `case 1` message says January.
