# 01 - Console

This lesson introduces the browser developer console and common `console` methods used while learning and debugging JavaScript.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the console examples.

## What This Covers

- Logging numbers, strings, booleans, variables, and multiple values with `console.log()`.
- Showing errors and warnings with `console.error()` and `console.warn()`.
- Displaying object data in a table with `console.table()`.
- Grouping related console output with `console.group()` and `console.groupEnd()`.
- Styling console messages with `%c` and a CSS string.

## Things To Notice

Open `index.html` in a browser, then open DevTools and look at the Console tab. The page itself is simple because the learning happens in the console output.
