# 09 - Working With Strings

This lesson introduces string concatenation, template literals, and common string properties and methods.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` experiments with string values and logs the final result.

## What This Covers

- Building strings with concatenation.
- Building strings with template literals and `${}` placeholders.
- Creating a string object with `new String()`.
- Reading `.length` and accessing characters by index.
- Inspecting the string prototype.
- Changing case with `.toUpperCase()` and `.toLowerCase()`.
- Finding and extracting characters with `.charAt()`, `.indexOf()`, `.substring()`, and `.slice()`.
- Cleaning and changing strings with `.trim()` and `.replace()`.
- Checking content with `.includes()`.
- Getting the primitive value with `.valueOf()`.
- Splitting a string into an array with `.split()`.

## Things To Notice

The script reassigns `x` many times, so the console shows only the final example. Move `console.log(x)` upward or add extra logs if you want to inspect each method one at a time.
