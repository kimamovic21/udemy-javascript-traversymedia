# 07 - Operators

This lesson introduces common JavaScript operators used for arithmetic, assignment, string concatenation, and comparison.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` assigns many different expressions to `x` and logs the final result.

## What This Covers

- Arithmetic operators: `+`, `-`, `*`, `/`, `%`, and `**`.
- String concatenation with `+`.
- Increment and decrement with `++` and `--`.
- Assignment operators such as `+=`, `-=`, `*=`, `/=`, `%=`, and `**=`.
- Loose equality and inequality with `==` and `!=`.
- Strict equality and inequality with `===` and `!==`.
- Greater than, less than, and related comparisons.

## Things To Notice

Strict comparisons are usually safer because they compare both value and type. For example, `2 === "2"` is false because one value is a number and the other is a string.
