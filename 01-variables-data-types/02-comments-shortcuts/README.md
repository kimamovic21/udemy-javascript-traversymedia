# 02 - Comments and Shortcuts

This lesson shows how comments work in JavaScript and how they can be used to explain code or temporarily disable code while testing.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains examples of single-line and multi-line comments.

## What This Covers

- Single-line comments with `//`.
- Multi-line comments with `/* ... */`.
- Commenting out code so it does not run.

## Things To Notice

The console output is intentionally quiet in this lesson because the example code is inside comments. The main idea is learning what JavaScript ignores when it runs.
