# 12 - Math Object

This lesson introduces the built-in `Math` object and common methods for calculations and random numbers.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` assigns different `Math` results to `x` and logs the final result.

## What This Covers

- Square roots with `Math.sqrt()`.
- Absolute values with `Math.abs()`.
- Rounding with `Math.round()`, `Math.ceil()`, and `Math.floor()`.
- Exponents with `Math.pow()`.
- Minimum and maximum values with `Math.min()` and `Math.max()`.
- Random decimals with `Math.random()`.
- Random whole numbers in a range by combining `Math.random()` and `Math.floor()`.

## Things To Notice

The final example creates a random integer from 1 to 100. Refresh the browser and watch the console output change.
