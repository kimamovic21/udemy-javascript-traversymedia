# 04 - Data Types

This lesson introduces JavaScript's primitive and reference data types and uses `typeof` to inspect values.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` defines sample values for each data type and logs the final inspected value.

## What This Covers

- Primitive values: string, number, boolean, null, undefined, symbol, and bigint.
- Reference values: arrays, objects, and functions.
- Checking a value's type with the `typeof` operator.
- The special `typeof null` behavior, where JavaScript reports `object`.
- Functions being reported as `function` by `typeof`.

## Things To Notice

Only the last `output` assignment is logged. Change which value is assigned to `output` if you want to inspect different examples in the console.
