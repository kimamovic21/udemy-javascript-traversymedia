# 10 - Capitalize Challenge

This challenge practices string indexing, string methods, and concatenation by turning `"developer"` into `"Developer"`.

## Files

- `index.html` loads the challenge page and includes `script.js`.
- `script.js` contains three possible solutions.
- `10-capitalize-challenge.md` contains the original challenge instructions, hints, and solution notes.

## What This Covers

- Reading the first character with `.charAt(0)` or `[0]`.
- Capitalizing one character with `.toUpperCase()`.
- Getting the rest of a string with `.substring()` or `.slice()`.
- Combining strings with `+` or a template literal.

## Things To Notice

All three solutions follow the same idea: uppercase the first character, keep the rest of the string unchanged, then join both parts into a new string.
