# 13 - Number Challenge

This challenge combines random numbers, arithmetic operators, and template literals to print calculation results.

## Files

- `index.html` loads the challenge page and includes `script.js`.
- `script.js` contains the implemented challenge solution.
- `13-number-challenge.md` contains the original challenge instructions, hints, and solution notes.

## What This Covers

- Creating random integers with `Math.random()` and `Math.floor()`.
- Storing arithmetic results in variables.
- Calculating sum, difference, product, quotient, and remainder.
- Formatting output strings with template literals.
- Logging each result to the console.

## Things To Notice

`x` is generated from 1 to 100 and `y` is generated from 1 to 50. Because both values are random, each browser refresh can produce different console output.
