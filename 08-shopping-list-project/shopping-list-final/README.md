# Shopping List Final

This folder contains the finished shopping list project from the Modern JavaScript From The Beginning course.

## Files

- `index.html` contains the final project markup.
- `style.css` contains the completed app styling.
- `script.js` contains the full shopping list behavior.
- `images/note.png` is the app header image.
- `images/screen.png` is the screenshot used in this README.

## What This Covers

- Adding and removing shopping list items.
- Clearing the full list.
- Filtering visible items by search text.
- Persisting items with local storage.
- Editing existing items and resetting the form state.
- Preventing duplicate entries.

## Things To Notice

- `init()` groups the event listener setup and starts the app in one place.
- `checkUI()` keeps the controls and form button synced with the current app state.
