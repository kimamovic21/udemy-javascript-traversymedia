# Starter Files

This folder contains the starting HTML, CSS, JavaScript, and image asset for the shopping list project.

## Files

- `index.html` sets up the form, list container, filter input, clear button, and Font Awesome icons.
- `style.css` provides the complete visual layout for the project.
- `script.js` is intentionally empty so the project logic can be built step by step.
- `images/note.png` is the note icon used in the page header.

## What This Covers

- Reviewing the project markup before adding JavaScript behavior.
- Connecting the stylesheet, script file, Font Awesome, and local image asset.
- Establishing the DOM elements that later lessons will select and update.

## Things To Notice

- The form uses `id="item-form"` and the input uses `id="item-input"`, which become the main entry points for the JavaScript.
- The clear button and filter input already exist in the HTML, but they do not do anything until later lessons.
