# Set Edit Mode

This lesson adds the first part of editing by selecting an existing list item and putting the app into edit mode.

## Files

- `index.html` contains the form, list, filter, and clear controls.
- `style.css` adds styling for the edit state.
- `script.js` tracks edit mode and updates the form button.
- `images/note.png` is the page header image.

## What This Covers

- Tracking edit state with `isEditMode`.
- Selecting a list item for editing.
- Applying and removing the `edit-mode` class.
- Changing the form button text, icon, and background color.

## Things To Notice

- Clicking a list item now fills the input with that item's text.
- The app can enter edit mode in this step, but the update-and-reset behavior is completed next.
