# 10 - Insert Elements

This lesson explores different ways to insert elements, text, and HTML around existing DOM nodes.

## Files

- `shopping-list/index.html` contains the shopping list page used for insertion practice.
- `shopping-list/script.js` contains the insertion examples.
- `shopping-list/style.css` styles the shopping list page.
- `shopping-list/images/` contains the image assets used by the page.

## What This Covers

- Inserting elements with `insertAdjacentElement()`.
- Inserting text with `insertAdjacentText()`.
- Inserting HTML with `insertAdjacentHTML()`.
- Inserting before an existing child with `insertBefore()`.
- Using positions like `beforebegin`, `afterbegin`, `beforeend`, and `afterend`.
- Creating new nodes before inserting them.

## Things To Notice

Only `insertElement()` is called by default. You can call the other functions one at a time to see how each insertion method places content differently.
