# 14 - Styles Classes

This lesson shows how to read, update, and toggle classes and inline styles from JavaScript.

## Files

- `index.html` contains the style and class demo page.
- `script.js` contains the class and style examples.
- `style.css` styles the demo page.

## What This Covers

- Reading all classes with `className`.
- Replacing classes with `className`.
- Reading classes with `classList`.
- Looping through `classList`.
- Adding, removing, toggling, and replacing classes.
- Updating inline styles through the `style` property.
- Applying styles to multiple list items.
- Running DOM changes from a button click handler.

## Things To Notice

The `run()` function only executes when the button is clicked. That makes this lesson interactive compared with the scripts that run immediately on page load.
