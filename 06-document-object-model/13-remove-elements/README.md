# 13 - Remove Elements

This lesson covers multiple ways to remove DOM elements from the page.

## Files

- `shopping-list/index.html` contains the shopping list page used for removal practice.
- `shopping-list/script.js` contains the removal examples.
- `shopping-list/style.css` styles the shopping list page.
- `shopping-list/images/` contains the image assets used by the page.

## What This Covers

- Removing an element directly with `remove()`.
- Removing a child from its parent with `removeChild()`.
- Selecting list items with `nth-child()`.
- Selecting list items from a NodeList by index.
- Writing multiple versions of a remove helper function.
- Using an arrow function for a compact helper.

## Things To Notice

The active code removes the clear button and the second list item. The other remove functions are left in place so you can compare different approaches.
