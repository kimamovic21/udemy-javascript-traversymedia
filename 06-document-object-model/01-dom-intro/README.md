# 01 - DOM Intro

This lesson introduces the Document Object Model and shows how JavaScript can access and change page content through the browser's `window` and `document` objects.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the DOM introduction examples.

## What This Covers

- Inspecting the global `window` object.
- Accessing the `document` object from `window`.
- Reading common document elements like `body` and `links`.
- Writing directly to the document with `document.write()`.
- Selecting an element with `getElementById()`.
- Selecting an element with `querySelector()`.
- Changing element content with `innerHTML` and `innerText`.

## Things To Notice

This lesson changes visible page content, not just console output. Open `index.html` in the browser and compare the page with the Console output.
