# 12 - Replacing Elements

This lesson demonstrates several ways to replace existing DOM elements and content.

## Files

- `shopping-list/index.html` contains the shopping list page used for replacement practice.
- `shopping-list/script.js` contains the replacement examples.
- `shopping-list/style.css` styles the shopping list page.
- `shopping-list/images/` contains the image assets used by the page.

## What This Covers

- Replacing an element with `replaceWith()`.
- Replacing element markup with `outerHTML`.
- Replacing multiple list items in a loop.
- Using a ternary expression while updating item HTML.
- Creating a new heading element.
- Replacing a child element with `replaceChild()`.

## Things To Notice

All replacement functions are called at the end of the script, so the page changes immediately when it loads.
