# 07 - Create Elements

This lesson introduces creating DOM elements with JavaScript and adding them to the page.

## Files

- `shopping-list/index.html` contains the shopping list page used for element creation practice.
- `shopping-list/script.js` contains the element creation examples.
- `shopping-list/style.css` styles the shopping list page.
- `shopping-list/images/` contains the image assets used by the page.

## What This Covers

- Creating an element with `document.createElement()`.
- Adding classes, ids, and attributes to a new element.
- Creating text with `document.createTextNode()`.
- Appending text to an element with `appendChild()`.
- Appending a new element to the page.
- Adding a new element inside the shopping list.

## Things To Notice

The example creates a `div` and appends it to the `ul`. That is useful for seeing the mechanics, even though a real shopping list item would normally be an `li`.
