# 01 - Event Listeners

This lesson introduces browser event listeners and shows different ways to respond when a user clicks the clear button.

## Files

- `shopping-list/index.html` contains the shopping list page used for event practice.
- `shopping-list/script.js` contains the event listener examples.
- `shopping-list/style.css` styles the shopping list page.
- `shopping-list/images/` contains the image assets used by the page.

## What This Covers

- Selecting a button from the DOM.
- Handling click events with `onclick`.
- Handling click events with `addEventListener()`.
- Using a named event handler function.
- Removing list items from the DOM.
- Removing an event listener with `removeEventListener()`.
- Triggering a click event from JavaScript with `.click()`.

## Things To Notice

Assigning `onclick` more than once replaces the previous handler, while `addEventListener()` can attach multiple handlers to the same event.
