# 06 - Input Events

This lesson covers form input events and shows how form controls can update the page as the user interacts with them.

## Files

- `shopping-list/index.html` contains the shopping list page used for input event practice.
- `shopping-list/script.js` contains the input, change, focus, and blur examples.
- `shopping-list/style.css` styles the shopping list page.
- `shopping-list/images/` contains the image assets used by the page.

## What This Covers

- Listening for `input` events on a text field.
- Listening for `change` events on a select input.
- Reading current control values from `e.target.value`.
- Reading checkbox state from `e.target.checked`.
- Responding to `focus` and `blur`.
- Updating the heading from input events.
- Changing input outline styles from focus handlers.

## Things To Notice

The text input updates the heading while you type, while the select input updates after its selected value changes.
