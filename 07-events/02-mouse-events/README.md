# 02 - Mouse Events

This lesson demonstrates common mouse and drag events by attaching handlers to the shopping list logo image.

## Files

- `shopping-list/index.html` contains the shopping list page used for mouse event practice.
- `shopping-list/script.js` contains the mouse event examples.
- `shopping-list/style.css` styles the shopping list page.
- `shopping-list/images/` contains the image assets used by the page.

## What This Covers

- Listening for `click` and `dblclick`.
- Listening for right-clicks with `contextmenu`.
- Handling `mousedown` and `mouseup`.
- Handling wheel events with `wheel`.
- Handling hover changes with `mouseover` and `mouseout`.
- Handling drag events with `dragstart`, `drag`, and `dragend`.
- Updating page colors from an event handler.

## Things To Notice

Double-clicking the logo toggles the page background and text colors. The other handlers log messages so you can see exactly when each mouse event fires.
