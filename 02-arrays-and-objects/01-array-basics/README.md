# 01 - Array Basics

This lesson introduces arrays as ordered lists of values and shows the first ways to create, read, and update them.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains the array basics examples.

## What This Covers

- Creating arrays with array literals.
- Creating arrays with the `Array` constructor.
- Storing mixed value types in an array.
- Reading values by index.
- Combining array values with other expressions.
- Reading the `.length` property.
- Updating existing array items by index.
- Adding new items by assigning to a specific index.
- Adding items to the end with `array[array.length]`.

## Things To Notice

Array indexes start at `0`, so `fruits[2]` reads the third item. The script keeps assigning different values to `x`, then logs the final result.
