# 07 - Destructuring and Naming

This lesson introduces shorthand object property names and destructuring for objects and arrays.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains shorthand property, object destructuring, nested destructuring, and array destructuring examples.

## What This Covers

- Using variable names as object property names.
- Reading object values with destructuring.
- Renaming destructured properties.
- Destructuring nested object properties.
- Destructuring arrays by position.
- Collecting remaining array values with the rest operator, `...`.

## Things To Notice

The `todo.id` property is destructured into a new variable named `todoId`. This is useful when the original property name is too generic or conflicts with another variable.
