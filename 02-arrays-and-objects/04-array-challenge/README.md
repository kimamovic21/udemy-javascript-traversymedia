# 04 - Array Challenge

This challenge practices the array methods from the previous lessons.

## Files

- `index.html` loads the challenge page and includes `script.js`.
- `script.js` contains implemented solutions for both array challenges.
- `04-array-challenges.md` contains the original challenge instructions, hints, and solution notes.

## What This Covers

- Mutating an array to change `[1, 2, 3, 4, 5]` into `[6, 5, 4, 3, 2, 1, 0]`.
- Using `.push()`, `.unshift()`, and `.reverse()`.
- Combining two arrays while removing a duplicate value.
- Solving the same problem with `.slice().concat()` or spread syntax plus `.splice()`.

## Things To Notice

The file keeps multiple solution approaches in the code. This is useful for comparing how different array methods can produce the same final result.
