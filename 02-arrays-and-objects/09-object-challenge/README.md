# 09 - Object Challenge

This challenge combines arrays, nested objects, property updates, destructuring, and JSON conversion.

## Files

- `index.html` loads the challenge page and includes `script.js`.
- `script.js` contains the implemented challenge solution.
- `09-object-challenge.md` contains the original challenge instructions and solution notes.

## What This Covers

- Creating an array of book objects.
- Giving each book a nested `status` object.
- Updating nested properties with dot notation.
- Destructuring and renaming a property.
- Converting an array of objects to a JSON string with `JSON.stringify()`.

## Things To Notice

The challenge updates `status.read` after the original objects are created. That is intentional practice for reaching into nested object data and changing a specific property.
