# 06 - Object Spread Operator and Methods

This lesson continues with objects by showing object construction, nested data, object merging, and useful object helper methods.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains examples of object spread and object methods.

## What This Covers

- Creating an object with `new Object()`.
- Adding properties after object creation.
- Reading deeply nested object properties.
- Merging objects with the spread operator.
- Merging objects with `Object.assign()`.
- Working with arrays of objects.
- Reading object keys with `Object.keys()`.
- Counting properties by using `Object.keys(obj).length`.
- Reading values with `Object.values()`.
- Reading key-value pairs with `Object.entries()`.
- Checking for a property with `.hasOwnProperty()`.

## Things To Notice

The spread operator and `Object.assign()` can both merge objects. In this lesson, both approaches create a combined object from `obj1` and `obj2`.
