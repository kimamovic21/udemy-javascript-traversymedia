# 05 - Object Literals

This lesson introduces object literals and shows how to read, update, delete, and add object properties.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains object literal examples.

## What This Covers

- Creating objects with key-value pairs.
- Storing nested objects and arrays inside an object.
- Accessing properties with dot notation.
- Accessing properties with bracket notation.
- Updating existing properties.
- Deleting properties with `delete`.
- Adding new properties after object creation.
- Adding methods to objects.
- Using `this` inside an object method.
- Working with property names that contain spaces.

## Things To Notice

Dot notation is clean when the property name is a normal identifier. Bracket notation is needed for names like `"first name"` because they contain a space.
