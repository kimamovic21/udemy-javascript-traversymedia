# 02 - Array Methods

This lesson explores common array methods for adding, removing, searching, slicing, splicing, and chaining.

## Files

- `index.html` loads the lesson page and includes `script.js`.
- `script.js` contains examples of common array methods.

## What This Covers

- Adding to the end with `.push()`.
- Removing from the end with `.pop()`.
- Adding to the beginning with `.unshift()`.
- Removing from the beginning with `.shift()`.
- Reversing an array with `.reverse()`.
- Checking for values with `.includes()`.
- Finding positions with `.indexOf()`.
- Turning arrays into strings with `.toString()` and `.join()`.
- Copying part of an array with `.slice()`.
- Removing or replacing items with `.splice()`.
- Chaining multiple methods together.

## Things To Notice

Some methods mutate the original array and some return a new value. `splice()` changes the original array, while `slice()` leaves the original array alone.
